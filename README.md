## Configuración del archivo .env

Para configurar el acceso a la base de datos, crea un archivo `.env` en la raíz del proyecto con las siguientes variables:

DB_HOST=localhost
DB_USER=tu_usuario
DB_PASSWORD=tu_contraseña
DB_NAME=softjobs
DB_PORT=5432
